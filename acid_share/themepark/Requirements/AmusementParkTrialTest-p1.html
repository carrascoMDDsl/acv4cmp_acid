Welcome to the CM Productions AMUSEMENT PARK trial test (page 1/2)

This page is about what we provide to you in this trial task.

Introduction

  • Imagine you are working for an amusement park with attractions and rides.
  • Imagine that the park is installing a huge screen near each attraction, or imagine that the park is doing a free mobile application.
  • In any case, the park wants to let the users know how long they should wait before riding an attraction.

What you get

  • You will get html+css+javascript code that already does something.
  • You will have to edit the javascript to achieve a certain goal.
  • The code gets static information from a database (hardcoded in your sample code) about:
    1. The positions of the attractions over a map.
    2. The roads connecting the attractions.
    3. The crossings between roads.
  • The code ALREADY paints the attractions, crossings and roads

Try the sample code

  • Get it now - Download it here.
  • Uncompress.
  • Open the index.html page from a browser.
  • You will see a map of the park with attractions.
    • The attractions, crossings and roads that are painted over the map with javascript.
    • The painting is done with an object oriented drawing library named KineticJS.
  • Open index.html in your favorite text editor. Have a look to the source, to get familiar with it.
